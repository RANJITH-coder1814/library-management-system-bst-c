# library-management-system-bst-c
A menu-driven Library Book Management System implemented in C using Binary Search Tree (BST). It supports book insertion, searching by ISBN, multiple tree traversals, and displays BST statistics and properties.

This project implements a **Library Book Management System** using a **Binary Search Tree (BST)** in C. Books are organized using ISBN numbers as keys for efficient searching, sorting, and traversal.
